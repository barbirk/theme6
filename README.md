# Arts et Math — Thème 6

A fun, bilingual (French/English) Progressive Web App to help an 11-year-old prepare for a math exam. Based on **Thème 6 — Arts et math!** covering Sections 26–30 plus Revision.

## Features

- **6 Interactive Lessons** with practice problems
- **7 Quiz Banks** (6 topic-specific + 1 comprehensive review)
- **8 Mini-Games** including memory, factor trees, fraction painting, decimal challenges, and geometry
- **Formula Sheet** with quick-reference cards
- **Progress Dashboard** with XP, streaks, and achievement badges
- **Bilingual** — switch between French and English
- **Dark/Light Mode** with automatic preference detection
- **PWA** — installable, works offline

## Tech Stack

- Pure HTML + CSS + Vanilla JavaScript (no frameworks)
- Service Worker for offline caching
- GitHub Pages for hosting

## Math Topics

| Section | Topic |
|---------|-------|
| 26 | Equivalent expressions, missing terms, prime factorization |
| 27 | Multiplying a whole number by a fraction |
| 28 | Decimal operations (×10, ×100, ×1000), decimal × decimal |
| 29 | Translation, reflection, friezes & tilings |
| 30 | Probability (fraction, decimal, percentage) |
| Revision | Comprehensive review + applied problem-solving |

## Development

Open `index.html` in any modern browser. No build step required.

## Deployment

Pushing to the `main` branch triggers automatic deployment to GitHub Pages via the included workflow.
